# github_final_project# 🌿 Paradise Nursery Shopping Application

## 📌 Project Name
Paradise Nursery Shopping Application

---

## 📖 Project Description
Paradise Nursery Shopping Application is a React-based web application that simulates an online plant shop. Users can browse a variety of indoor plants, add them to a shopping cart, manage quantities, and view the total cost dynamically.

This project is developed as part of the **"Developing Front-End Apps with React"** course to demonstrate React fundamentals such as components, props, state management, and event handling.

---

## 🌱 Features
- Display a collection of indoor plants
- Add plants to shopping cart
- Increase or decrease quantity of items
- Remove items from cart
- Automatically calculate total price
- Clean and responsive user interface

---

## 🛠️ Technologies Used
- React.js
- JavaScript (ES6+)
- HTML5
- CSS3
